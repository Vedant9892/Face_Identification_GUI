IMPORTANT

Step1: Enter images of the person in the folder named 'person1', 'person2', etc.
Step2: Enter the name and age of the person in the file named 'person1.txt', 'person2.txt', etc.

After doing this, run the train_faces.py file to train the model.

This will create a folder person1 in trained models folder. 
Which will contain mathematical data of the person (encodings.npz)

Now you can run the gui_app.py file to start the live recognition.

you can create more folders for more people.
